# Day 4 Commands Practice

## Hour 1 - Python setup

### python3 --version
Checks installed Python version.

### python3 hello.py
Runs the Python script.

### What I observed
Python runs scripts using `python3 filename.py`.

## Hour 2 - Variables and functions

### python3 variables_demo.py
Runs a script showing variables and data types.

### python3 function_demo.py
Runs a function example.

### What I observed
Python uses indentation and functions are defined using `def`.


## Hour 3 - File handling

### python3 file_read.py
Reads the full file content.

### python3 file_line_read.py
Reads the file line by line.

### What I observed
Python can read files easily using `with open(...)`.

## Hour 4 - Error handling

### python3 error_demo.py
Shows how Python handles missing file error.

### What I observed
`try/except` prevents the script from crashing and lets us handle errors cleanly.
