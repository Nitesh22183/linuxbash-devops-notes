# Day 5 Notes - Python Logic

## if, elif, else
These are used to make decisions in Python.

- `if` checks the first condition
- `elif` checks another condition if the first one is false
- `else` runs if none of the above conditions are true


## What a loop means

A loop repeats code.

A `for` loop is useful when you want to go through:
- numbers
- a list of tools
- lines in a file
- server names
- URLs

## for loop
A for loop is used to repeat code for each item in a list or range.

Example:
for tool in tools:
    print(tool)

## while loop
A while loop runs as long as the condition is true.

It is important to update the variable inside the loop, otherwise the loop may never stop.
