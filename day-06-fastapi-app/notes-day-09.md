# Day 09 Notes - AWS EC2 Deployment

## Region
A geographic AWS area, such as ap-south-1.

## Availability Zone
An isolated data center inside a region.

## VPC
A logically isolated private network in AWS.

## Subnet
A smaller network inside a VPC.

## Internet Gateway
A component that enables internet access for resources in public subnets.

## Route Table
A set of rules that controls where network traffic is directed.

## Security Group
A virtual firewall that controls inbound and outbound traffic to an EC2 instance.
